# Getting Started with the React Splitter Component

**Repository Description**  
This repository contains a quick‑start React sample that demonstrates how to integrate and configure the Syncfusion React Splitter component in a React application.

A quick‑start project that helps you create and configure the Syncfusion React Splitter component. This project includes example code to configure core features such as resizing, collapsing, and expanding panes, as well as rendering nested split panes within a React layout.

## Project Overview
The purpose of this project is to help developers understand the basic usage and configuration of the Syncfusion React Splitter component. It demonstrates how to divide application content into resizable sections to build flexible and user‑friendly layouts.

## Features
- Integration of the Syncfusion React Splitter component  
- Resizable, expandable, and collapsible panes  
- Support for nested splitter panes  
- Flexible layout management for React applications

## Prerequisites
Make sure that your development environment meets the following requirements:
- Node.js (latest version recommended)  
- Visual Studio Code  

## Installation and Running the Application
1. Clone the repository.
```bash
git clone https://github.com/SyncfusionExamples/getting-started-with-the-react-splitter-component.git
```
2. Open the project directory in Visual Studio Code.
3. Install all required dependencies using:`npm install`.
4. Run the project using:`npm start`.
5. Open your browser and navigate to the URL shown in the terminal (typically `http://localhost:3000`).
